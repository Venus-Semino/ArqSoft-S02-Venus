# Ahorcado 
Un juego sencillo escrito en C#

En este juego se encuentra una clase dios en dónde no tiene una estructura, por ende será modificada para que tenga una mejor estructura. En este código se identificaron los siguientes violaciones de SOLID:
- SRP: la clase juego implementa más de una responsabilidad.
- DIP: las palabras está en el código inicial, haciendolo más dificil.
- OCP: hay que modificar cada vez que se intenta agregar un nuevo juego.
